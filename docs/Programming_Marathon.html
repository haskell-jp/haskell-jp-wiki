Attention, Please

2004/07/31 09:46:39 JST –nobsun

Sampou.Org は回線が細いので、Haskell:Marathon では臨時に、(株)タイムインターメディア様のサーバを利用することにします。

http://openspace.timedia.co.jp/~nobsun/marathon.cgi にアクセスしてください。

どんな風にやろうか？

  • Programming Haskell に、課題ごとのページを用意
  • 課題に着手前に、その課題のページをチェック
  • 未着手なら、そのページでとりかかることを宣言してから着手
  • とりあえず、動作するコードができたらそのコードを貼り付ける
  • 貼り付けられたコードをみんなでリファクタリング
  • リファクタリングの過程がわかるように、古いコードは残す
  • 議論はページ上、IRC(誰かチャネルを用意してくださいな)、その場で、行なう
  • ページ上以外でおこなわれた議論はあとでできるだけ、ページに転記する
  • 30分に一度程度の頻度で、IRCなどに進捗を報告
