Session 28 Exercise 1 - Test 5 は宿題

  • ここか mixi で解く？
    • こことCategoryTheory_圏論勉強会ノートで解きましょう。
  • Session 28 Exercise 1 (a)
    • category of pointed sets では 0 = 1 なので 0×A = 1×A = A。A として例えば２要素からなる pointed set をとると 0×A = A の要素数は２、一方 0 = 1 の要素数は１なので isomorphic ではない。
  • Session 28 Exercise 1 (b)
    • B1 = 1、B2 = 1、A を２要素からなる pointed set とすると、A×B1 ＋ A×B2 = A＋A の要素数は３。一方 1＋1 = 1 なので A×(B1＋B2) = A×1 = A の要素数は２。よって A×B1 ＋ A×B2 と A×(B1＋B2) は isomorphic ではない。
