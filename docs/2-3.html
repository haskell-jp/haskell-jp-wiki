地図を更新し続けた方がいいかも

GHCの全体像がわかりにくいので、勉強会を開く毎に地図と各地のキーとなる概念を更新して置いといた方がいいかも。 そうすれば、途中参加者も地図を見て参加しやすくなる。 (by @khibino )
