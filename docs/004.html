みんなの宿題

関数

merge :: Ord a => [a] -> [a] -> [a]

は、整列されたリストを2つ取り、1つの整列されたリストにして返す関数である。以下に使用例を示す。

> merge [2,5,6] [1,3,4]
[1,2,3,4,5,6]

関数mergeを再帰を用いて定義せよ。ただし、関数insertやisortなど、整列されたリストを処理する関数は利用してはならない。
