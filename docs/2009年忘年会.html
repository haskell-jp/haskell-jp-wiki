2009年忘年会

Haskell 2009年末の集い(忘年会前哨戦)

  • 日時：2009年12月22(火) 18:30〜20:00
  • 場所：IIJ 大会議室2 (17F)
    • 張り紙に従って大会議室2まで来て下さい。
    • エレベータは2カ所にあります。張り紙のある方を使って下さい。

忘年会

  • 日時：2009年12月22(火) 20:00〜22:00
  • 場所：シャヒダワット
  • 金額：4,500円ぐらい

割り当て

自分でテーマを1つ選んで、右に名前を書いて下さい。早い者勝ちです。

  • “Haskell 2010”
    • DoAndIfThenElse (kazu)
    • HierarchicalModules (shelarcy)
    • EmptyDataDeclarations (nobsun)
    • FixityResolution
    • ForeignFunctionInterface
    • LineCommentSyntax (nakanowatari)
    • PatternGuards (heita)
    • RelaxedDependencyAnalysis (sakai)
    • LanguagePragma (yamamiya.takashi/propella)
    • NoNPlusKPatterns
  • “GHC 6.12.x”
    • parallel execution (maoe)
    • ThreadScope
    • Dynamic linking
      • きちんと動くのは Linux 環境なので、できれば Linux を常用している方に選んでもらいたいです。
    • Unicode (nwn)
      • 元ネタ
    • ABI
    • Tuple Sections (oneda)
      • Python を使ったことがある人に、どう便利なのか教えてもらいたいです。
    • Record puns（、Record wildcards、Record field disambiguation）
      • 正確には GHC 6.12.1 で入ったというわけではなくて、これらの機能に対する拡張・修正が行われただけですが、GHC 6.10.x の頃にはあまり（日本で）話している人はいなかったと思うので

自己紹介も含めて、5分の発表にまとめてきて下さい。VGA プロジェクターを用意します。

調べたいテーマ、聞きたい話を追加するも OK です。
