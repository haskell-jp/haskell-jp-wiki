Session 26

  • linear category
    • p280. sum of maps が実際にテキスト通りの形になっているかの確認。板書1, 板書2, 板書3
      • matrix の sum になっている部分以外のみを取り出す操作をして、その式の一部が identity matrix と α（identity matrix の inverse)の合成＝identity と同等になる事を利用している。
  • Exercise 1 … はタイムアップ、宿題。
    • αがnaturalである事を仮定して、functorとnatural transformationを利用した解答。
      • しかし、このテキストではfunctorとか登場しないので、もっと簡単な解答があるはず…？
      • ちなみに、natural transformation の inverse なので、natural なのは当たり前。
      • 解答の筋は同じですが functor を表にださずに整理したものをCategoryTheory_圏論勉強会ノートの Session 26 Exercise 1 に書きました。
  • Exercise 2
    • 解答
  • zero map があって、initial object / terminal object が存在しない例はある？
    • 例の１つ。object 間に自然数（0含む）のラベルがついたmapが自然数個存在
    • nのラベルのmap と、mのラベルのmapのcompositionはn×mになる
    • で、zero map は0のラベルがついたmap
  • object は１つでよい。つまり、モノイドでよい。
