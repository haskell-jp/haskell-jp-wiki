宿題

filter関数と同じ動きをする

filter' :: (a -> Bool) -> [a] -> [a]

という関数をfoldlかfoldrを使って実装してください。

リスト内包表記の場合のfilter’の実装は↓になると思います。

filter' p xs = [ x | x <- xs, p x]

再帰を使った場合、fiter’の実装は↓になると思います。

filter' _ [] = []
filter' pred (x:xs)
  | pred x = x : filter pred xs
  | otherwise = filter pred xs
